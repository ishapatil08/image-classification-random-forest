# Image Classification using Random Forest

## Project Description
This project implements Image Classification using a supervised learning approach.
A Random Forest classifier is trained on labeled image data to classify images into target categories.

## Steps Followed
1. Data Collection
2. Image Preprocessing (Resizing, Flattening)
3. Train-Test Split
4. Model Training using RandomForestClassifier
5. Model Evaluation
